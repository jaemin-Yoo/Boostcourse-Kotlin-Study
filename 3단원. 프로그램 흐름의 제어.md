## if ~ else 문의 간략화
```kotlin
var max: Int
if (a > b)
  max = a
else
  max = b
```
```kotlin
val max = if (a > b) a else b
```

### 블록과 함께 사용

```kotlin
val max = if (a > b) {
    println("a 선택")
    a // 마지막 식인 a가 반환
} else {
    println("b 선택")
    b
}
```

### 범위 연산자
- 변수명 in 시작값..마지막값
- score in 80..89이면 score 범위에 80부터 89까지 포함
